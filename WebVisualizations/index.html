Web Development Project

Project Overview

This project's scope includes using HTML, CSS and Bootstrap to develop a webpage that demonstrates the knowledge gathered in class regarding these tools. In this webpage, the change in weather conditions such as temperature and humidity will be graphed by variating the latitude in which they are measured. The results will be showcased in 6 different pages, which can be accessed through the following links:

  • Max Temperature
  • Humidity
  • Cloudiness
  • Wind Speed
